# 011-favorites-alerts

## Goal
Deliver favorites, my teams, and alert workflows.

## Build
1. Build favorite actions for matches, teams, and competitions across all major pages.
2. Build the favorites page and left-rail `My Teams` module.
3. Implement alert subscriptions for kickoff, goals, cards, period changes, and final result where feed support exists.
4. Add guest-local persistence plus account sync for favorites and alert settings.
5. Add homepage and board prioritization rules based on favorites and recent views.
6. Add analytics for favorite creation, alert opt-in, and favorites-driven return usage.

## Done When
- Favorites feel native across the entire app.
- Alert settings can be saved and respected.
- Personalized navigation mirrors the reference product's behavior.
