# 001-scope

## Goal
Freeze MVP boundaries, stack, and non-goals from `app-write-up.md`.

## Build
1. Lock product scope to football-first MVP: auth, live/fixtures/results/tables, match detail, odds display, insights, basic slips, paid creator content, moderation, responsive UI, i18n, dark mode.
2. Confirm architecture: Next.js App Router + JavaScript + Prisma + MySQL + Styled Components + Redux Toolkit + i18n.
3. Define roles: guest, user, creator, moderator, admin.
4. Add explicit non-goals for MVP: AI recs, multi-sport, native apps, deep historical analytics.
5. Publish a single source of truth document in repo for scope sign-off.

## Done When
- Scope sign-off exists.
- MVP and later features are clearly separated.
- Team agrees on stack and role model.
