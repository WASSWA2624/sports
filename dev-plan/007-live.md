# 007-live

## Goal
Implement live match and results experience.

## Build
1. Build live feed with status filters and quick league pivots.
2. Build match detail modules: timeline, stats, lineups, key events.
3. Implement controlled client refresh windows for live fixtures.
4. Add result snapshots and post-match data freeze behavior.
5. Add loading/skeleton/error/empty states for all live views.

## Done When
- Live updates are timely and stable.
- Match detail is complete for supported fixtures.
- Result views remain consistent after final whistle.
